💎 CORE TELEMETRY

High-fidelity structural features that set My.Jarvis apart

🛡️

100% Offline Speech

Equipped with a compact Vosk ASR acoustic engine. Voice recognition is computed entirely on your hardware with no internet required and zero remote data storage.

🤖

Neural Wake-word

Listens passively in an ultra-lightweight thread for "Hey Jarvis" triggers using openwakeword, ensuring prompt wake responsiveness with low energy footprint.

🔊

Live Audio Metrics

Computes mathematical root-mean-square (RMS) microphone amplitudes in real-time, feeding a highly fluid, customizable graphic visualizer wave interface.

⚡

Windows Automation

Includes ctypes background browser focusing and start-menu shortcut indexing, letting you launch software and manage media globally without changing active windows.

system_commands.cfg
"open [app]" // Index and launch Windows applications
"start [game]" // Searches Start Menu and executes game shortcuts
"launch chrome" // Forces browser launch
"open cmd" // Triggers Windows Command Prompt
"run notepad" // Quick launcher utilities
"play song [song name]" // Launches specific Spotify song (confirms yes/no)
"stop music" // Pauses Spotify or system media playback
"play music" // Resumes audio playback
"next music" // Skips to next track
"previous music" // Skips back to previous track

"reduce youtube volume" // Focuses background window and reduces volume 10%
"search [query]" // Registers query and prompts verification
"look up [topic]" // Searches topic on Google (requires confirmation)
"find [topic]" // Performs Google search in browser

🛠️ SYSTEM CORE ARCHIVE

How the system compiles, deploys, and updates

1. Install Development Libraries
pip install customtkinter pyaudio vosk openwakeword numpy keyboard requests
2. Compile Standing Executable (.exe)
pyinstaller --onefile --noconsole --name="Jarvis" --add-data "model;model" --collect-all vosk --collect-data customtkinter --collect-data openwakeword ui.py
3. One-Click Compile Batch Command (build.bat)
@echo off
pyinstaller Jarvis.spec
pause
